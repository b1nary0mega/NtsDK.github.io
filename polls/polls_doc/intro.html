О системе Ассистент-выборы¶

Ассистент-выборы - что это?

Система «Ассистент выборы» предназначена для фиксации информации о явке избирателей по часам и сравнения явки между выборами. Система реализована в виде интерактивной веб-страницы. Все, что вам нужно для работы с системой «Ассистент выборы» это веб-браузер. Проверялась работа программы в Firefox, Chrome. Работа в Internet Explorer не проверялась. Для работы системе «Ассистент выборы» не требуется соединение с интернетом.

Порядок работы с Ассистент-выборы

  1. Открываете Ассистент-выборы
  2. Загружаете базу для редактирования
  3. Вносите изменения
  4. Сохраняете базу для последующего запуска

Ассистент-выборы не может работать напрямую с файлом, поэтому сохранение не выполняется сразу в файл. Это необходимо делать отдельно. На закрытии вкладки с Ассистент-выборы всегда показывается напоминалка о необходимости сохранения базы. В процессе работы с Ассистент-выборы все изменения сразу сохраняются в странице. Например, при изменении текста описания выборов на первой странице, как только вы завершите ввод, текст будет сохранен. Сохраненный файл базы является обычным текстовым файлом фиксированной структуры.

Технические подробности в двух словах

Система «Ассистент выборы» написана на языке JavaScript с использованием библиотек jQuery (календарь для ввода дат), Chart.js (круговые диаграммы), Bootstrap 3 (элементы дизайна), Ramda (элементы функционального программирования), Select2 (списки с поиском) и др. База данных хранится в формате json.

Авторы используемых иконок с сайта www.flaticon.com: Anton Saputro, Pavel Kozlov, Budi Tanrim, Catalin Fertu, Picol и Freepik. Автор иконки сортировки FatCow.

Исходный код системы по открыт под лицензией Apache 2.0. Последняя версия НИМСа находится в репозитории https://github.com/NtsDK/polls-assistant. Исходники документации доступны по ссылке https://github.com/NtsDK/polls-assistant-doc.

Специальные обозначения

курсив - термины. Моноширинный шрифт - названия кнопок и элементов страниц.
